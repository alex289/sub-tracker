# Sub Tracker

Simple and _local_ first subscription tracker.

## Development

### Setup

> Requires Node.js 22.x, pnpm 9.x and Cargo 1.83.x

```sh
pnpm install

cd src-tauri
cargo install
```

### Run

```sh
pnpm tauri dev
```

### Build

```sh
pnpm tauri build
```
